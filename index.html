Insert-Sort Demo

This is a visual demonstration of the insert sort method.

Try it out:

The insert method will sort each element in the list one by one. Starting at the index of 1, the function will take the current index and compare it to the element before it. If the value of the previous element is lower, then the current element will be inserted before the previous one. When it finds a value that it is equal to or greater than, it will stop in its current position. Once every element has been checked, the list will be in ascending order.

Big O = O(n2)
